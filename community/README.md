## \ud83d\udce5 Download Instructions

### Option 1: Clone the Repository
```bash
git clone https://github.com/seanshin0214/persona-hub-gpt.git
cd persona-hub-gpt/community
```

### Option 2: Download Individual Files
Click on each `.txt` file and click \"Raw\" then \"Save As\"

### Option 3: Download as ZIP
1. Go to main repository page
2. Click green \"Code\" button
3. Select \"Download ZIP\"
4. Extract and navigate to `community/` folder

## \ud83d\udcdd File Format

Each persona file contains:
- **Metadata**: Author, category, version, license
- **Core Identity**: Who this persona is
- **Expertise Areas**: Specific knowledge domains
- **Communication Style**: How they interact
- **Use Cases**: When to activate this persona
